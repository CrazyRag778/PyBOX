PYOS(1)                     User Commands                    FLEXFETCH(1)

NAME
    flexfetch - Display system hostname in stylized ASCII art banner

SYNOPSIS
    flexfetch

DESCRIPTION
    This program reads system information from a JSON file and
    prints a formatted ASCII art banner displaying the hostname.

FILES
    /imp/system.json
    JSON file containing system information.
    Must include:
        {
            "HOSTNAME": "your-host-name"
        }

OUTPUT
    A multi-line ASCII banner including:
        - Stylized "PYOS" text
        - The system hostname
        - Tagline: "(Power . Your . OS)"

ERRORS
    FileNotFoundError
        If "../../imp/system.json" does not exist.

    KeyError
        If "HOSTNAME" is missing from the JSON file.

    JSONDecodeError
        If the JSON file is improperly formatted.
